gnrc_onehot2therm

Convert N bit ONEHOT code to N-1 bit THERMOMETER code.
for example:
0000_0001 -> 000_0000
0000_0010 -> 000_0001
0000_0100 -> 000_0011
0000_1000 -> 000_0111
…
1000_0000 -> 111_1111
For the sake of a simple circuit structure, we don’t consider illegal input.
This module instantiate a gnrc_lzc_therm, see more in gnrc_lzc_therm.sv
